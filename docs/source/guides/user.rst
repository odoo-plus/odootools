User guide
==========

Overview
--------

Odoo-tools provide a few command lines that can be used to manage
your odoo projects. It can be used for multiple things such as configuring
odoo configuration, discovery paths that contains odoo addons, initializing
an odoo database, installing or uninstalling odoo addons and more.
